# 🏋️ Gym Management System

A web-based Gym Management System built using ASP.NET Core MVC, designed to manage members, subscriptions, and daily gym operations with a clean and maintainable architecture.

---

## 🚀 Features

- 🔐 User Authentication & Role-Based Authorization  
- 👤 Member Management (Add, Update, Delete)  
- 💳 Subscription Management  
- 📊 Dashboard for managing gym operations  
- 🧱 Clean and structured MVC architecture  

---

## 🛠️ Tech Stack

- ASP.NET Core MVC  
- Entity Framework Core  
- SQL Server  
- LINQ  
- Identity (Authentication & Authorization)  
- Layered Architecture  
- Repository & Unit of Work Patterns
